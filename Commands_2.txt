1.Как вывести список установленных пакетов утилиты -tree по дате установки:
cat var/log/dpkg.log |grep "install" | grep tree

2.Как посмотреть дату и время обновления пакетов утилиты tree:
stat $(which tree)

3.Как вывести список установленных пакетов и пронумеровать его:
dpkg -l | cat -n
dpkg -l | nl

4.Как вывести ошибки из файла syslog:
sudo grep -i error /var/log/syslog

5.Как посмотреть время и дату удаленных пакетов из dpkg.log:
grep remove /var/log/dpkg.log

6.Как посмотреть изменения пакетов в dpkg.log за дату и показать строки со второй по пятую:
grep "2024-07-07.*status installed" /var/log/dpkg.log | sed -n '2,5p'
grep "2024-07-07.*status installed" /var/log/dpkg.log | sed -n 2,5p

7.Как посмотреть последние 50 строк, +4 до и 4 после и пронумеровать их из history.log:
tail -n50 /var/log/history.log | cat -n | sed -n -e 46,49p -e 51,54p

8.Как посмотреть список запущенных процессов и отфильтровать их по пользователю:
ps -u
ps -fu <user_name>
ps -eF | sort -k l

9.Как проверить возможность доступа к порту 22:
sudo telnet localhost 22
nc -zvw3 localhost 22

10.Что делает данный конвеер du -hs /var/log/* | sort -rh :
   Данный конвеер выводит в порядке убывания размеры всех файлов, хранящихся в папке (директории) /var/log/

   du (Disk Usage - использование диска) - команда Linux, применяется для вывода размера всех файлов, находящихся в определённой папке
   -h (human-readable) - флаг (опция) для вывода размера в единицах измерения, удобных для человека (К, М)
   -s (summarize) - флаг (опция) для вывода только общего размера
   /var/log/ - путь к папке (директории)
   * - символ для выбора всех файлов, хранящихся в директории
   sort - команда Linux, применяется вывода текстовых строк в определенном порядке, т.е. для сортировки
   -r (recursion) - флаг (опция) для сортировки текстовых строк в обратном порядке
   -h (human-readable) - флаг (опция) для вывода размера в единицах измерения, удобных для человека (К, М)
